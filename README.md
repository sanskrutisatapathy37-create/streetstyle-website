# StreetStyle 👕✨
StreetStyle is a modern and responsive clothing store website created using HTML, CSS, and JavaScript.  
It is designed to provide a stylish and smooth online shopping experience.

## 🚀 Features
- Responsive design for mobile, tablet, and desktop
- Attractive homepage layout
- Product showcase section
- Smooth navigation
- Modern UI design

## 🛠️ Built With
- HTML5
- CSS3
- jQuery

## 📌 Purpose
This project was built to improve my frontend development skills and create a real-world e-commerce website design.
